# 📚 API de Estudiantes

API REST creada con **Node.js** y **Express** para realizar operaciones CRUD sobre estudiantes.

## 🚀 Tecnologías

* 🟢 Node.js
* ⚡ Express.js
* 📄 JavaScript

## ▶️ Ejecutar

```bash
npm install
npm start
```

📍 **Servidor:** `http://localhost:8000`

## 📌 Endpoints

* 📥 **GET** `/students` → Obtener mensaje de prueba.
* ➕ **POST** `/create` → Crear un estudiante.
* ✏️ **PUT** `/update/:id` → Actualizar un estudiante.
* 🗑️ **DELETE** `/delete/:id` → Eliminar un estudiante.

## ✅ Validación

Campos obligatorios al crear un estudiante:

* 👤 firstName
* 👤 lastName
* 🎂 age
* 📧 email
